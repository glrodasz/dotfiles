# Vim

Vim config and solarized colors from `~/dotfiles/vim`.

## Prerequisites

One-time clone of vim-colors-solarized:

```bash
cd ~/dotfiles/vim
git clone git@github.com:altercation/vim-colors-solarized.git
```

## Install

```bash
make slink-vim
```

Links `~/.vimrc` and `~/.vim/colors/solarized.vim`.
